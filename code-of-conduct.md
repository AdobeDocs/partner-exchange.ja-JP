# アドビの行動規範

## アドビの誓約

オープンで快適な環境を育むために、私たちはコントリビューターおよび管理者として、私たちのプロジェクトとコミュニティへの参加を、年齢、身体を問わず、すべての人にとってハラスメントのない体験にすることを誓います
大きさ、障害、民族、性自認および表現、経験レベル、国籍、外見、人種、宗教、または性自認および性的指向。

## アドビの基準

ポジティブな環境の創出に貢献する行動の例
の一般的な使用法には次が含まれます。

* 快適で差別のない表現を使用する
* 異なる視点および経験を尊重する
* 建設的な批判を謹んで受け入れる
* コミュニティにとって最善なことに注力する
* 他のコミュニティメンバーに対して共感を示す

参加者による許容されない行動の例を以下に示します。

* 性的な言葉や画像の使用と歓迎されない性的注意
アドバンス
* 挑発、侮辱的／軽蔑的コメント、個人的攻撃や政治的攻撃
* 公的または個人的なハラスメント
* 物理的または電子的な他人の個人情報を公開する
明示的な許可なしにアドレス
* 合理的に不適切と考えられる他の行為
プロフェッショナルな設定

## アドビの責任

プロジェクトメンテナーは、許容される行動の基準を明確にする責任を負い、許容されない行動があった場合は適切かつ公正な是正措置をとることとします。

プロジェクト管理者は、この行動規範に沿わないコメント、コミット、コード、Wiki編集、イシュー、その他の投稿を削除、編集、却下する権利および責任を有します。または、一時的に禁止する権利または責任を有します。
不適切、脅威的、攻撃的、または有害と見なされるその他の行動の貢献者を永続的に管理します。

## 範囲

個人がプロジェクトまたはそのコミュニティを代表している場合、本行動規範はプロジェクトスペース内と公開スペース内の両方に適用されます。 プロジェクトやコミュニティを代表する例には、プロジェクトの公式メールを使用することが含まれます
住所、公式ソーシャルメディアアカウントでの投稿、オンラインまたはオフラインのイベントでの担当者の役割が可能です。 プロジェクトを代表する行動について、プロジェクトメンテナーからさらに詳細な定義および解釈が提示される場合があります。

## 実施

虐待、嫌がらせ、その他の方法で許容できない行動が発生した場合は、プロジェクト チーム（`Grp-opensourceoffice@adobe.com`）に連絡して報告することができます。 すべての苦情は審査および調査され、状況に応じて必要かつ適切と見なされる対応が行われます。 プロジェクトチームは、インシデントの報告者に関する秘密を守る責任があります。 特定の実施ポリシーの詳細は別途投稿される場合があります。

本行動規範を誠実に遵守または実施しないプロジェクトメンテナーに対しては、プロジェクトを統率する他のメンバーの判断により、一時的または恒久的な措置が講じられる場合があります。
